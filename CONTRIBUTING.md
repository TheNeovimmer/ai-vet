# Contributing to AI Doctor

Thanks for your interest in contributing.

## How to Contribute

1. Fork the repo at [github.com/TheNeovimmer/ai-vet](https://github.com/TheNeovimmer/ai-vet)
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'Add some feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request

## Guidelines

- Keep changes focused and well-scoped
- Write clear commit messages
- Update docs if adding/changing features
- Test your changes before submitting

## Issues

Report bugs and request features via [GitHub Issues](https://github.com/TheNeovimmer/ai-vet/issues).
